# Feature Roadmap: LockedIn-Pro

## Phase 1: Core Redesign (Pillbox & Bento)
- [ ] Migrate Dashboard to Bento Grid layout.
- [ ] Implement Pillbox component library (Buttons, Cards, Inputs).
- [ ] Update Typography to Outfit/Inter.
- [ ] Liquid animation for Compliance Progress.

## Phase 2: Missing Essential Features
- [ ] **Financial Stakes (The Forfeit)**: Integration for commitment-based penalties (API hooks for Beeminder).
- [ ] **Adaptive Mandates**: System logic that scales task difficulty based on visual tension/micro-expression markers and Journal sentiment.
- [ ] **Progression System (The Rank)**: 
    - XP for Compliance.
    - Tiered ranks (e.g., Novice, Compliant, Devoted, LockedIn).
    - Unlockable UI skins (Themed variations of the Bento grid).
- [ ] **Ambient Accountability**:
    - Tray app/Widget for real-time lock status.
    - Browser extension that restrict sites when Integrity Factor ($I$) drops.

## Phase 3: The Gaze & The Submission
- [ ] **Levels of Possession**: Selection of intensity tiers from [Toy] to [Property].
- [ ] **The Gaze**: Implementation of the background mobile camera loop for spontaneous Inspections.
- [ ] **Search-Augmented Penance**: The AI Master performs live web searches to devise unique training methodologies.
- [ ] **Spontaneous Demands**: Background notification engine for randomized "Chastity Checks."

## Phase 4: Permanent Indoctrination
- [ ] Expanded AI visual analysis for real-time compliance monitoring.
- [ ] High-resolution camera stream integration for constant multi-factor visual validation.
